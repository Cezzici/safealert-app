# SafeAlert – Disguised Mobile Application for Domestic Violence Support

**SafeAlert** is a disguised mobile app that allows victims of domestic violence to silently send alerts by interacting with a simple Tic Tac Toe game. The system includes:

- 🧠 A Java-based frontend (desktop/mobile) built with Gluon Mobile and JavaFX
- 🛰️ Native device features access using Gluon Charm Down (GPS, vibration, file write)
- 🌐 A PHP & MySQL backend for alert reception and processing
- 🧾 Web dashboard for authorities and NGOs to manage cases and generate PDF reports

## 🎯 Purpose

This project was developed as part of a Bachelor's thesis at the Romanian-American University, aiming to provide a discreet, scalable, and fully documented way to support victims through technology.

## 📂 Components

### ✅ App (Frontend)
Repository: [safealert-app](https://github.com/Cezzici/safealert-app)  
A JavaFX-based application that masks the alert functionality as a game.

### ✅ Server (Backend)
Repository: [safealert-server](https://github.com/Cezzici/safealert-server)  
Handles alerts, form generation, intervention tracking, and PDF exports (via FPDF).

## 🧪 Technologies Used

- Java 17, JavaFX
- PHP 7.4+, MySQL 8.0, Apache, FPDF
- IntelliJ IDEA, XAMPP, GitHub

## 📊 Features

- Silent alert transmission after specific game gestures
- Automatic alert processing and geolocation
- Role-based web access: admin, authority, NGO
- PDF export for forms and interventions
- Local simulation and testing enabled
